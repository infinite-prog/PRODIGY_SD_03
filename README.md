# PRODIGY_SD_03
Simple Contact Management Program - Task 03 | Prodigy Infotech Software Development Internship

**Objective:**  
Develop a console-based program that allows users to store and manage contact information. Users can add new contacts and view the contact list.

## Features Implemented
- Add a new contact (Name, Phone Number, Email)
- View all contacts with proper formatting
- Simple menu-driven interface
- Input validation for menu choices
- Separate `Contact` class for better code organization
